This project provides 3 separate graphs of information on a test subject for the bacteria found in their navels. The horizontal bar chart shows the top 10 OTU IDs (Operational taxonomic unit) that were found on the test subject. The bubble chart shows all of the OTU IDs and provides the bacterias found when the cursor hovers over the bubbles. The gauge in the top right corner shows how the frequency of the test subject washing their navel. There is a dropdown menu that can be used to select any of the test subjects.
